什么是 IPv6?

IPv6 是新一代互联网协议，提供 128 位地址空间，解决 IPv4 地址不足问题，并提升端到端连通性。

海量地址空间

可分配地址远大于 IPv4。

端到端连接

设备可直接使用公网可路由地址。

路由效率

协议更现代，便于网络优化。

安全可控

结合防火墙可实现更细粒度控制。

双栈 Dual Stack

同时支持IPv4和IPv6，这是目前最理想的配置，可以访问所有网站

仅IPv4 IPv4 Only

只支持IPv4，这是目前最常见的配置，可以访问绝大多数网站

仅IPv6 IPv6 Only

只支持IPv6，较少见的配置，部分仅IPv4的网站可能无法访问

无连接 No Connection

网络连接异常或防火墙阻止

如何启用IPv6？

  • 运营商支持：联系你的ISP（互联网服务提供商）确认是否提供IPv6服务
  • 路由器配置：在路由器管理界面中启用IPv6（DHCPv6或SLAAC）
  • 操作系统：现代操作系统（Windows 10+、macOS、Linux）默认支持IPv6
  • 防火墙：确保防火墙不阻止IPv6流量

IPv6 性能优化建议

DNS 优化

优先使用延迟更低的 IPv6 DNS。

MTU 调整

原生 1500，PPPoE 可尝试 1492，隧道可尝试 1280。

协议优先级

对比延迟后决定优先 IPv4 还是 IPv6。

常见问题排查指南

IPv6 没地址

  1. 确认运营商已开通 IPv6。
  2. 检查光猫和路由器 IPv6 开关。
  3. 终端执行 ipconfig /all 或 ip -6 addr。

IPv6 速度慢

  1. 排查是否走隧道协议。
  2. 更换 DNS、调整 MTU。
  3. 对比 ping -4 与 ping -6。
